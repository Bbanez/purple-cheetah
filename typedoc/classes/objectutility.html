Static combine

  • combine(master: object, slave: object, level?: string): any
    • Defined in src/util/object.util.ts:21

    Combine 2 objects into 1 following few rules:

    1. If master does not have a key from a slave, it will be added.
    2. If slaves key value is NULL, it will be ignored.
    3. All key values from a master that are not present in a slave will stay unmodified.
    4. Array key values from a slave will be just copied to master, even if it is array of objects.
    5. If key of a slave is an object, recursion will occur.

    Parameters

    • master: object

      Object that will be modified.

    • slave: object

      Object which key values will be used.

    • Optional level: string

    Returns any
