Controller

  • Controller(uri?: string): (Anonymous function)
    • Defined in src/decorators/controller.decorator.ts:37

    Decorator that will annotate a Class as a Controller width all required properties. Class annotated with Controller decoration can parsed as a new instance to Application decorator.

    Example

     
     // my-controller.ts
 @Controller('/my-controller')
 export class MyController {
   @Get()
   async getMethod() {
     return 'This is my controller.'
   }
 }
     //app.ts
 @Application({
   port: 1280,
   controllers: [ new MyController() ],
   middleware: [],
   exceptionHandlers: []
 })
 export class App {};

    After starting the application route GET: /my-controller will be exposed. See Controller Methods for more information about Get decorator.

    Parameters

    • Optional uri: string

    Returns (Anonymous function)
